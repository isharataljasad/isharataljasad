RULES / HARD

FEEDBACK / LOOP
T0خط أساسأعراض + تحاليل + إيقاع + أدوية + سياق عضو.

T2سحب الدعماختبار قدرة الجسد بعد خفض الدعم أو إيقافه.
T3قفل الوقايةتثبيت العادات والفحوص الدورية لمنع الحلقة نفسها.
